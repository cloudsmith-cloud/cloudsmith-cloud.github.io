What we're building

Product Roadmap

From MVP to a full-lifecycle platform. CloudSmith is built in phases — each one shippable, each one useful on its own.

Phase IV — MVP

The foundation. Everything that comes after builds on this.
In Development
Platform kernel — core, identity, secrets
Cluster management and inventory
Azure Local deployment pipelines
S2D sizing and planning tools (Surveyor absorb)
REST API, SDK, CLI, and PowerShell module
Standalone on-premises deployment model
Azure PaaS deployment model
Hybrid deployment model with bidirectional sync
Web portal with module-contributed SPA

Phase V — First Operations Layer

Day-two operations: monitoring, hardware, updates, and workloads.
Planned
Dell iDRAC / Generic Redfish hardware management
Azure Monitor Workspace metrics + DCR pipelines
VictoriaMetrics for standalone on-prem monitoring
OS update orchestration (Azure Update Manager, LCM)
AI-assisted troubleshooting
Workload management — VMs and templates
Self-hosted image library (ISOs, VHDs)
Optional Azure image connectors (Marketplace, Compute Gallery)
MSP multi-tenancy
Runner payload library

Phase VI — Ecosystem Expansion

Broader hardware support, Hyper-V clusters, and discovery.
Planned
Bidirectional on-prem ↔ PaaS sync engine
Hyper-V cluster deployment (standalone, WSFC)
Workload discovery and assessment
As-built documentation generation
Lenovo XCC hardware support
HPE iLO hardware support

Post-MVP — Advanced Capabilities

Security, governance, integrations, and expanded workload coverage.
Roadmap
Security posture — Defender for Servers, CIS scanning, vulnerability management
Governance and policy-as-code
FinOps and cost reporting
ITSM integration — ServiceNow
Backup visibility — Commvault, Veeam
AKS on Azure Local
Azure Virtual Desktop (AVD + FSLogix)
VMware/Nutanix migration tooling
Decommissioning and retirement workflows
Workload optimization and right-sizing
Lightweight on-host agent (cs-agent)
Terraform / Pulumi / Bicep IaC provider

Future — Later Releases

Parking lot items. No committed timeline — evaluated after core platform matures.
Parking Lot
Multi-cloud resource inventory (AWS/GCP via Azure Arc connectors)
MCP Server (Model Context Protocol integration)
FedRAMP / IL4 compliance posture
Bare-metal PXE provisioning
SaaS managed offering
