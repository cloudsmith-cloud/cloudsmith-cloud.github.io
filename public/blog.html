CloudSmith v1.0.0-preview1 is available for private preview

We're excited to announce that CloudSmith v1.0.0-preview1 is now available for our first cohort of private preview participants.

What is CloudSmith?

CloudSmith is an open-source cluster management platform for Hyper-V and Azure Local environments. It covers the full infrastructure lifecycle: deployment, cluster management, workload operations, update management, hardware lifecycle, monitoring, and AI-assisted diagnostics.

What's in preview1?

  • Online installer — one-command install on Windows Server 2022/2025 with Hyper-V
  • PaaS deployment — Bicep templates targeting Azure Container Apps
  • First-run wizard — no pre-configuration required; guided setup on first access
  • Cluster enrollment — enroll Hyper-V clusters via the CloudSmith Agent
  • Management portal — React + Fluent UI portal with cluster, workload, and health views

How to join the preview

Private preview is by invitation only. If you're interested in joining, email preview@cloudsmith.cloud with a brief description of your environment (OS version, cluster size, use case).
