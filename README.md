# cloudsmith-cloud.github.io

Public website for the CloudSmith open-source project — hosted at [cloudsmith.cloud](https://cloudsmith.cloud).

CloudSmith is a modern cloud platform management solution for Azure Local and Hyper-V environments, covering the full infrastructure lifecycle from deployment through AI-assisted diagnostics.
